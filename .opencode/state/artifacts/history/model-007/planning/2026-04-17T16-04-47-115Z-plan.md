# Planning Artifact — MODEL-007: Generate sword-projectile via Blender-MCP

## 1. Scope

Generate a low-poly sword-projectile 3D model via blender-agent MCP and export to `assets/models/sword-projectile.glb`. The sword-projectile is a thrown sword for ranged attacks, targeting ≤200 triangles per CANONICAL-BRIEF.md.

## 2. Scene Description

**Model**: Low-poly thrown sword (projectile)
- Single blade + guard + minimal handle
- Elongated proportions for aerodynamic thrown-weapon silhouette
- No complex crossguard detail — simplified for ≤200 tri budget
- Color/material: Steel blade (silver/gray), dark handle wrap

**Design targets**:
- Triangle count: ≤200 (hard cap)
- 1 PBR material (SteelGray for blade, DarkWrap for handle — or single unified material)
- Manifold mesh, outward-facing normals
- Godot-compatible GLB export

## 3. Triangle Budget Strategy

| Part | Est. Tris |
|------|-----------|
| Blade (elongated box/plane) | ~120 |
| Guard (thin cross bar) | ~40 |
| Handle (small cylinder) | ~30 |
| Pommel (tiny sphere) | ~10 |
| **Total** | **~200 (at cap)** |

Budget is tight — no decorative bevels, no complex guard flourishes. Every primitive must be purpose-built.

### Step 2 — project_initialize (new clean scene)
```
Tool: blender_agent_project_initialize
Args:
  project_root: /home/rowan/womanvshorseVC

  units: METRIC
  factory_startup: null
```
**Chaining note**: output_blend becomes input_blend for Step 3.

### Step 3 — scene_batch_edit (geometry build)
```
Tool: blender_agent_scene_batch_edit
Args:
  project_root: /home/rowan/womanvshorseVC
  input_blend: /tmp/sword-projectile-init.blend
  output_blend: /tmp/sword-projectile-geom.blend
  operations:
    - action: create_primitive
      primitive: plane
      name: Blade
      properties:
        dimensions: [0.05, 0.8, 0.01]  # thin, elongated blade
    - action: set_transform
      object: Blade
      properties:
        location: [0, 0.4, 0]
        rotation: [0, 0, 0]
    - action: create_primitive
      primitive: cube
      name: Guard
      properties:
        dimensions: [0.3, 0.04, 0.04]  # cross guard bar
    - action: set_transform
      object: Guard
      properties:
        location: [0, 0.0, 0]
    - action: create_primitive
      primitive: cylinder
      name: Handle
      properties:
        vertices: 6
        radius: 0.025
        depth: 0.25
    - action: set_transform
      object: Handle
      properties:
        location: [0, -0.2, 0]
    - action: create_primitive
      primitive: uv_sphere
      name: Pommel
      properties:
        radius: 0.04
        segments: 6
    - action: set_transform
      object: Pommel
      properties:
        location: [0, -0.35, 0]
    - action: join
      objects: [Blade, Guard, Handle, Pommel]
```
**Chaining note**: output_blend becomes input_blend for Step 4.

### Step 4 — material_pbr_build (single material)
```
Tool: blender_agent_material_pbr_build
Args:
  project_root: /home/rowan/womanvshorseVC
  material_name: SwordMaterial
  input_blend: /tmp/sword-projectile-geom.blend
  output_blend: /tmp/sword-projectile-mat.blend
  assignments: [SwordProjectile]  # joined mesh name
  maps:
    base_color: [0.75, 0.75, 0.8, 1.0]  # steel gray
    roughness: 0.4
    metallic: 0.9
```
**Chaining note**: output_blend becomes input_blend for Step 5.
