## Root Cause

**BLENDER-MCP-CHAIN (systemic):** The Blender MCP bridge does not forward `input_blend`/`output_blend` parameters from the JSON payload to the Blender command line. Every `scene_batch_edit` call runs with `--factory-startup`, starting from a fresh Blender scene, making scene chaining impossible regardless of how parameters are named or passed.

This is the **same root cause** documented in the original MODEL-003 implementation artifacts (superseded at `2026-04-17T03:35:22-518Z` and `2026-04-17T03:44:07-881Z`) and the MODEL-004 review artifact (`2026-04-11T16:22:46-995Z`).
