## Script Design Soundness

- **XZ-plane movement**: `velocity.x = dir.x * speed`, `velocity.z = dir.z * speed`, Y left at 0 — correct for fixed top-down orthographic camera.
- **NavigationAgent3D integration**: `_nav_agent.is_target_reachable()` guard before movement — correct Godot 4.6 pattern.

- **Contact damage**: `body.has_method("take_damage")` guard — correct pattern for targeting player or other damageable.
- **External navigation target**: `set_navigation_target(pos)` called by spawner — correctly decoupled from base class.
