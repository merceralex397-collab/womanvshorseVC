## What was implemented

### 1. AudioManager singleton (autoload)

**File:** `scripts/audio/audio_manager.gd`

A new autoload singleton registered in `project.godot` as `AudioManager`. All sounds are procedurally generated at init using `AudioStreamGenerator` + `AudioStreamGeneratorPlayback._mix_buffer()`. No external audio files.

**Architecture:**
- 5-slot pooled `AudioStreamPlayer` design ‚Äî each sound type gets one persistent player
- One `AudioStreamGenerator` per slot, filled once with all PCM samples via `_mix_buffer()`
- Ring buffer pre-fill on first play (`slot.played` guard), so subsequent plays are instant
- All sounds generated once in `_ready()` ‚Üí `_pool` array, zero per-play allocation

**Sound types:**
| Key | Description | Generation |
|---|---|---|
| `attack` | White-noise whoosh, exponential decay (~0.13 s) | `randf()` noise √ó exp envelope |
| `hurt` | Descending sine sweep 300‚Üí135 Hz (~0.20 s) | Phase-accumulated `sin()` with frequency sweep |
| `death` | Noise+90 Hz tone mix, very fast decay (~0.15 s) | Blended noise + sine |
| `victory` | Ascending C5-E5-G5-C6 arpeggio with ADSR (~0.52 s) | 4-note jingle at 22050 Hz |
| `defeat` | Descending G4-E4-C4 arpeggio (~0.50 s) | 3-note descending jingle |

**Public API:**
```
AudioManager.play_attack()
AudioManager.play_hurt()
AudioManager.play_death()
AudioManager.play_victory()
AudioManager.play_defeat()
AudioManager.play_jingle(victory: bool)
```

**File:** `scripts/attack_system.gd`

- `hit` signal connected to `_on_hit_audio` in `_ready()` (line 14)
- `_on_hit_audio` calls `AudioManager.play_attack()` on every successful hit (line 31)

### 3. Signal wiring ‚Äî player hurt

**File:** `scripts/player_controller.gd`

- `take_damage()` calls `AudioManager.play_hurt()` on every player hit (line 88)

### 4. Signal wiring ‚Äî enemy death

**File:** `scripts/enemies/horse_base.gd`

- `_on_enemy_died()` calls `AudioManager.play_death()` before `died.emit()` (line 96)

### 5. Signal wiring ‚Äî victory/defeat jingle

**File:** `scripts/ui/game_over_screen.gd`

- `_ready()` calls `AudioManager.play_jingle(victory)` after setting labels (line 20)
