## 3. Scene Structure Design

```
title_screen.tscn (Control root)
└── CenterContainer
    └── VBoxContainer
        ├── Label (game title: "WOMAN VS HORSE")
        └── Button (Start Game)
```

- **Control root:** Full-screen container, anchors to full rect
- **CenterContainer:** Centers content vertically and horizontally
- **VBoxContainer:** Stacks title label above button with spacing
- **Label:** Large stylized text for "WOMAN VS HORSE" — font size ~48, white/yellow color
- **Button:** "Start Game" text, minimum 48px height for touch-friendliness, connected to script handler

## 4. Script Design

**title_screen.gd** — extends `Control`

| Element | Detail |
|---------|--------|
| `_ready()` | Connect button `pressed` signal to `_on_start_game_pressed` |
| `_on_start_game_pressed()` | Calls `get_tree().change_scene_to_file("res://scenes/arena/arena.tscn")` |
| Optional animation | Title Label scale/fade tween on `_enter_tree()` for visual polish |

Input action "ui_accept" (already defined in project.godot for CORE-004) will allow keyboard/gamepad Enter to also trigger the button.
