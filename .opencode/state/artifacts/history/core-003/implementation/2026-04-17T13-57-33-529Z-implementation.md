# Implementation: CORE-003 — Wave Spawner (3D Positions)

## Summary

Implementation complete — all 5 ACs PASS. Created wave spawner scene and script with 5-wave progressive difficulty, randomized arena edge spawning, and full signal chain.

## Files Created

### `scripts/wave_spawner/wave_spawner.gd` (5353 bytes)

- State machine: `IDLE → SPAWNING → WAVE_ACTIVE → WAVE_CLEARED → ALL_COMPLETE`
- 5-wave progressive difficulty via `wave_configs` Array[Dictionary]
- `signal wave_started(wave_index, enemy_count)`, `signal wave_cleared(wave_index)`, `signal all_waves_complete`
- `_random_arena_edge_position(radius)` — `randi() % 4` selects edge, `randf_range(-radius, radius)` perpendicular axis, Y=0.0
- `alive_enemies` counter wired to `HorseBase.died` signal
- `enemy_scene: PackedScene` export for `horse_base.tscn` instantiation
- `player_ref: Node3D` export for NavigationAgent3D target
- Fallback `_enemy_container` resolution via `get_tree().get_first_node_in_group("arena")`
- `SpawnTimer` one-shot timer between waves

### `scenes/wave_spawner/wave_spawner.tscn` (308 bytes)

- Node3D root named `WaveSpawner` with `wave_spawner.gd` attached
- `SpawnTimer` Timer child with `one_shot = true`

### `scenes/arena/arena.tscn` (updated)

- Added `wave_spawner.tscn` as `WaveSpawner` child of `Arena` Node3D
- `load_steps` incremented from 2 → 3 to accommodate new external resource
